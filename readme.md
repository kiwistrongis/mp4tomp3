mp4tomp3
========
A little script that for converting mp4s to mp3s
------------------------------------------------

By: Kalev Kalda Sikes
https://github.com/KiwiStrongis/mp4tomp3

## Installation

Install vlc first :)

```bash
sudo ln -s mp4tomp3 /usr/bin/mp4tomp3
sudo chmod 755 /usr/bin/mp4tomp3
```
